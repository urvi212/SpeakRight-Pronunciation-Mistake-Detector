**Overview**

SpeakRight is an AI-powered pronunciation evaluation system designed to help users improve their English speaking skills. The system analyzes spoken audio, detects pronunciation mistakes, and provides real-time feedback to help learners correct their speech.

The goal of this project is to make pronunciation practice more interactive and accessible using speech processing and AI technologies.



**Key Features**

Detects pronunciation mistakes in spoken words or sentences

Provides feedback on incorrect pronunciation

Uses speech-to-text and phoneme-level comparison

Noise reduction for better speech detection accuracy

User-friendly interface for practicing pronunciation


**Technologies Used**

Python

Whisper AI (Speech Recognition)

Text-to-Speech (TTS)

Speech Processing

SequenceMatcher

Flask (for web interface)


**How It Works**

The user records or uploads speech audio.

The system converts speech to text using Whisper AI.

The spoken output is compared with the expected sentence.

Phoneme-level analysis identifies pronunciation mistakes.

Feedback is generated to guide the user toward correct pronunciation.
